# Paradigm Integrated Field Services (IFS) 4.0

![Version](https://img.shields.io/badge/version-11.9.0-blue)
![Tech Stack](https://img.shields.io/badge/tech--stack-React%20%7C%20Vite%20%7C%20Supabase%20%7C%20Capacitor-green)

Professional ERP and Onboarding system for Paradigm Office, featuring high-performance web and native mobile applications.

## 🚀 Quick Start

### 1. Prerequisites
- **Node.js** (v18 or higher)
- **npm** (v9 or higher)

### 2. Installation
```bash
npm install
```

### 3. Environment Setup
Copy the example environment file and fill in your credentials:
```bash
cp .env.local.example .env.local
```

### 4. Development
This project runs a dual-server setup (Frontend + Backend Proxy):
```bash
npm run dev
```

## 🛠️ Main Tech Stack
- **Frontend**: React 18, Tailwind CSS 4, Framer Motion
- **Tooling**: Vite 7, TypeScript 5
- **Backend/DB**: Supabase (PostgreSQL), Prisma, ts-node
- **Mobile**: Capacitor 7, Native WebViews
- **AI**: Google Gemini Pro (Document extraction & Verification)

## 🏗️ Project Structure
- `/src`: Core logic and server entry points.
- `/components`: UI library (HR, Inventory, Analytics, etc.).
- `/services`: API integration layers (Supabase, GenAI, Offline cache).
- `/android` & `/ios`: Native mobile projects.
- `/public`: Static assets and auto-generated service workers.
